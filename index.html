About

Shiloh Schedule — current site

Shiloh Schedule is a lightweight, browser-based schedule and countdown app for schools. It shows the active period, a live countdown timer, and provides flexible schedule management with user-customizable visuals and persistent settings.

Key features

  • Live countdown for the current period with automatic schedule switching.
  • Multiple schedules (Normal, Chapel, Pep Rally, Custom) with rename/save capability.
  • Appearance controls: background image or gray-themed panels, gradients, color picks, and timer shadow options.
  • Show period times option (Settings → Schedule) to display start/end times beside each period.
  • Auth & sync via Firebase (Google sign-in) to save settings across devices—optional and controlled by the user.
  • Chrome extension integration to sync gradient themes when available.

How to use

  1. Open Settings (cog) → Appearance to customize visuals or set a background.
  2. Use Settings → Schedule to switch schedules or enable "Show period times".
  3. Sign in with Google if you want settings synced across devices (optional).

Privacy & data

Personal data is only used for authentication and optional settings sync via Firebase. No data is sold. See the Privacy Policy for details.
